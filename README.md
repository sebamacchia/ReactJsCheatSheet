## What is?

- React is a JavaScript library for building user interfaces


```html
<tag style="property: value;"></tag>
```
- Embedded Style

```html
<head>
  <style type="text/css">
    selector {
      property: value;
    }
  </style>
</head>
```

- External Style Sheet

```html
<head>
  <link rel="stylesheet" type="text/css" href="style.css" />
</head>
```

## Syntax

```css
selector {
  property: value;
  }
}
```
